# Página web - Casa de eventos

Este es mi proyecto de página web llamado **Casa de eventos**, desarrollado con HTML, CSS y JavaScript.  
Está pensado para mostrar información y funcionalidades de una página web estática profesional.

## Tecnologías utilizadas

- HTML  
- CSS  
- JavaScript  

## Descripción del proyecto

Esta página web incluye:

- Página principal informativa  
- Menú de navegación  
- Secciones con contenido visual  
- Diseño básico y estructura clara  
- Interactividad con JavaScript  

## Cómo usar el proyecto

1. Descarga o clona este repositorio  
2. Abre el archivo `index.html` en tu navegador  
3. La página se visualizará automáticamente

## Qué pude aprender con este proyecto

- Estructura básica de páginas web  
- Uso de HTML, CSS y JavaScript  
- Organización de archivos  
- Publicación en GitHub Pages
